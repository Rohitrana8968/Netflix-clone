Netflix Clone

This project is a Netflix homepage clone built using HTML, CSS. The goal of this project is to practice frontend development by recreating the layout and design of a popular streaming platform.

It focuses on improving skills in responsive design, layout structuring, and basic.

Features  

Responsive Netflix-style landing page

Video background section

Movie/series thumbnail rows

Hover effects on movie cards

Clean and modern UI design

Mobile and desktop compatibility


Technologies Used

HTML5 – For structuring the webpage

CSS3 – For styling and layout

Git & GitHub – For version control

Project Objective

The main objective of building this project was to:

Practice frontend development skills

Understand how large websites structure their UI


Future Improvements

Add user authentication (login/signup)

Integrate a movie API for dynamic content

Add a backend using Node.js

Implement a fully functional streaming interface
